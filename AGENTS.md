# Agent Working Rules

## Project Names
Project 1: Investment AI Agent
Project 2: Investment AI Agent Intent

## General Rules
- Do not delete files unless explicitly instructed.
- Do not overwrite working code without explaining why.
- Do not hardcode API keys.
- Use environment variables.
- Always preserve the two-project comparison design.
- Always add tests for critical modules.
- Always update README when adding major features.

## Project 1 Rules
Investment AI Agent must remain a clean baseline:
- No AI Intent Classifier
- No OpenRouter model routing
- No Google ADK multi-agent orchestration
- Use one large investment prompt
- Use one selected LLM model
- Log chat, token, cost, and latency

## Project 2 Rules
Investment AI Agent Intent must include:
- AI Intent Classifier
- Domain Router
- OpenRouter LLM Gateway
- Google ADK Agents
- Token Monitor
- Cost Monitor
- Billing Ledger
- Chat History
- Evaluation support

## Safety Rules
- Never guarantee investment returns.
- Never say “buy now”, “sell now”, or “profit guaranteed”.
- Always include investment risk disclaimer.
- Ask for risk profile before personalized investment recommendation.
