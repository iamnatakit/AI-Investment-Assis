# Evaluation Plan

## Metrics
1.  **Token Consumption**: Total tokens (prompt, completion, reasoning) used per session/message.
2.  **Cost**: Total cost (USD/THB) per session/message.
3.  **Latency**: Time taken to generate a response (in milliseconds).
4.  **Quality**: Accuracy and appropriateness of the investment advice provided.

## Methodology
- Run a standardized set of test prompts against both Project 1 and Project 2.
- Record all metrics in the `usage_logs` and `billing_ledger` tables.
- Generate comparison reports highlighting the efficiency gains of the optimized architecture (Project 2).
