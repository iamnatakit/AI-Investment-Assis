# Project Brief

## Goal
Create the initial repository structure for an AI Investment Chatbot Token Optimization research project.

## Context
We will build two comparable projects to analyze token usage, latency, and cost:
1. **Investment AI Agent**: baseline single large prompt chatbot.
2. **Investment AI Agent Intent**: optimized chatbot with AI Intent, OpenRouter, Google ADK, token monitoring, chat history, and billing history.

## Scope
- Focus on evaluating token optimization and multi-agent architecture.
- Maintain a clear separation between the baseline and optimized projects to ensure accurate comparison.
