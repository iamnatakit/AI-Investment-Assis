# Security Rules

- **No Guarantees**: Never guarantee investment returns.
- **Restricted Phrases**: Never say "buy now", "sell now", or "profit guaranteed".
- **Disclaimers**: Always include an investment risk disclaimer in responses.
- **Profiling**: Always ask for a risk profile before providing personalized investment recommendations.
- **Data Protection**: Do not hardcode API keys or sensitive information in the codebase. Use environment variables.
